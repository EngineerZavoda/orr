# OpenRouter Request Scanner

## Простой Python-инструмент для поиска и тестирования бесплатных моделей OpenRouter.

Скрипт автоматически:

* находит доступные бесплатные модели
* отправляет тестовый запрос каждой модели
* измеряет время ответа
* показывает, какие модели работают
* определяет самую быструю
  
## Возможности

* ⚡ Параллельные запросы (быстрая проверка)
* 🧠 Авто-определение параметров модели (7B, 13B, 70B)
* ⏱ Замер времени ответа
* 📊 Удобный табличный вывод
* 🏆 Автоматический выбор самой быстрой модели

## 📦 Установка

1. Клонировать репозиторий

```
git clone https://github.com/YOUR_USERNAME/openrouter-models-scanner.git
cd openrouter-models-scanner
```

2. Создать виртуальное окружение

```
python3 -m venv venv
source venv/bin/activate
```

3. Установить зависимости


```
pip install -r requirements.txt
```

## 🔐 Настройка

Создай файл .env в корне проекта:

OPENROUTER_API_KEY=your_api_key_here

## ▶️ Запуск

```
python request.py
```

## 🧠 Как это работает

1. Получаем список моделей через OpenRouter API
2. Фильтруем только бесплатные
3. Отправляем тестовый запрос к каждой модели
4. Замеряем время ответа
5. Выводим результаты и выбираем лучшую модель
